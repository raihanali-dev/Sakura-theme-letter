# 🌸 Sakura Memory Garden

> *"Some memories are not written. They bloom."*

Sakura Memory Garden is an interactive storytelling experience built using pure HTML, CSS, and JavaScript. Inspired by Japanese cherry blossom gardens, anime-inspired scenery, and handwritten memories, the project transforms a simple webpage into a peaceful digital journey through letters, blossoms, and emotions.

Rather than functioning as a traditional website, Sakura Memory Garden is designed to feel like an artistic experience where visitors explore hidden letters beneath a blooming Sakura tree while the environment evolves around them.
